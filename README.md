# 运行步骤
  必备步骤：
  1. 运行爬虫项目
  2. 运行python-api项目
  3. 运行node-api转发项目
  4. 运行本项目
  
  本项目步骤：
  1. git clone https://github.com/qianbin01/lagou_vue
  2. cd lagou_vue
  3. npm install/yarn install
  4. npm run dev/npm start
  5. 浏览器访问http://localhost:8085
